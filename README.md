# ZooNak 🐾

Um projeto front-end interativo focado em explorar a rica biodiversidade do Brasil. 

## 💻 Sobre o projeto

O ZooNak foi desenvolvido para apresentar informações sobre a fauna brasileira de forma engajadora. A aplicação conta com uma interface moderna e funcionalidades dinâmicas para melhorar a experiência do usuário.

### ⚙️ Funcionalidades

- **Sistema de Abas:** Navegação fluida e organizada entre os diferentes conteúdos do site.
- **Quiz Dinâmico:** Um jogo de perguntas e respostas com sistema de pontuação em tempo real para testar os conhecimentos do usuário.
- **Agendamento Interativo:** Um formulário completo para agendamento, incluindo calendário integrado e validação de dados para garantir que as informações sejam preenchidas corretamente.

## 🛠️ Tecnologias Utilizadas

As seguintes ferramentas e tecnologias foram usadas na construção do projeto:

- **HTML5** - Estruturação semântica do conteúdo.
- **CSS3** - Estilização e responsividade.
- **JavaScript** - Lógica do quiz, validação de formulários e interatividade (sistema de abas).
- **Slick Carousel** - Biblioteca utilizada para criação de sliders na interface.
